# BHSM Minimal FeynRules Local Runner

These scripts are local execution helpers for a machine that has Mathematica
and FeynRules installed.

They are not evidence that FeynRules validation, UFO export, or loadability
has passed. Run them locally only after intentionally enabling and validating
`models/feynrules/BHSM_Minimal_Collider_Interface.fr`.
