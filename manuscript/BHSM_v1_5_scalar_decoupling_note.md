# BHSM v1.5 Scalar/Topographic Decoupling Note

BHSM v1.5 strengthens the scalar/topographic decoupling scaffold without
changing frozen predictions.

The sufficient scaffold conditions are:

1. exactly one light Higgs projection is allowed;
2. every orthogonal scalar/topographic mode is heavy, derivative-filtered,
   curvature-filtered, or screened;
3. a screened scalar/topographic mode is not counted as a new on-shell light
   particle in the state ontology;
4. any unscreened direct-coupled light scalar is a falsifier or open scalar
   risk.

The current proxy inventory passes these scaffold checks and contains zero
forbidden/open current scalar modes. Conditional filtered/screened modes remain
conditional, not fully proven safe.

This note does not prove scalar/topographic decoupling from the complete
internal action. The full scalar spectrum, topographic screening mechanism, and
action-level decoupling theorem remain open.
