# Projector Domain/Codomain Selection

Status: `OPEN_MISSING_PROJECTOR_DOMAIN_CODOMAIN_SELECTION`.

BHSM has independent charged-sector projector evidence, but the bounded CKM interface term does not contain a `Pi_u`/`Pi_d` sandwich or declare an operator domain and codomain. The one-way, adjoint-pair, sector-self, full-endomorphism, and maximal-self assignments therefore remain competing spaces.

Projector arithmetic alone does not derive the CKM exponent.
