# BHSM Author Ontology v0.8

The machine-readable source is
`artifacts/BHSM_author_ontology_v0_8.json`.

It fixes four interpretation rules for the minimal-action evaluator:

1. BHSM modes are physical boundary-local fields.
2. CP is the artifact-backed CP/Z6 boundary holonomy constraint; the standalone
   production vertex is a retired target.
3. `X_ch` is a conditional charged boundary-response operator.
4. Neutrino BHSM mass is a conditional propagation-locked curvature response,
   not an asserted ordinary static rest-mass matrix.

Artifact-backed and author-conditional evidence remain distinct. The ontology
uses no empirical derivation inputs and changes no frozen predictions. It does
not establish numerical closure or FeynRules/UFO/MadGraph readiness.
