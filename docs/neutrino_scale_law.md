# BHSM Neutrino Scale Law

The no-fit boundary package supplies dimensionless
`tau = 1/(4 pi^(3/2))`. It scales the dimensionless propagation response.

The repository contains no artifact-backed, non-calibrated conversion from
that response to eV or GeV. The scale status is therefore
`OPEN_MISSING_NEUTRAL_SCALE`.

Neither the electron-neutrino upper limit nor the W mass is used to fill this
gap. Reference values are comparison inputs only and are never theorem inputs.
