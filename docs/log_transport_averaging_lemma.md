# Log-Transport Averaging Lemma

Status: `ARTIFACT_BACKED_MATHEMATICAL_LEMMA`.

For fixed `sum x_i=L`,

`sum x_i^2 - L^2/N = sum (x_i-L/N)^2 >= 0`.

The unique minimizer is `x_i=L/N`, so multiplicative transport contributes `Z^(1/N)` per abstract equivalent channel.

The abstract log-transport averaging lemma does not by itself derive the CKM exponent.
