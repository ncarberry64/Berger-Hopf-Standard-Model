# Runtime Parameter Dependency Table

Machine-readable artifact:

```text
artifacts/BHSM_runtime_parameter_dependency_table_v0_9.json
```

The table distinguishes `BHSM_PURE_NOFIT` from `BHSM_COLLIDER_INTERFACE`.
Pure no-fit mode uses no empirical runtime inputs. Collider-interface mode may
use runtime empirical values only as simulation/comparison inputs. Runtime
inputs are not derivation inputs and may not modify BHSM constants, boundary
coefficients, mixing matrices, or frozen predictions.
