# Neutrino basis and scale theorem closure attempt

The boundary package supplies `K_nu`, its dimensionless coefficients, and a
PMNS mixing artifact. It does not supply a map from boundary channels to
physical neutrino states, a dimensionful scale, an ordering policy, a
Dirac/Majorana convention, or an observable map.

The attempt returns `OPEN_EXACT_MISSING_THEOREM`. It never reads an
electron-neutrino limit, PDG value, or other empirical reference as a theorem
input.

The neutral operator kernel is not promoted to a physical neutrino mass matrix without a physical basis, dimensional scale, and Dirac/Majorana convention.

Reference values, including PDG values, are comparison inputs only and are never theorem inputs.
