# Phase Three-H Bounded Blocker Resolution

BHSM Phase Three-H performs bounded resolution attempts for three Phase
Three-G vertex-table blockers:

- `X_ch` interaction operator;
- neutrino basis and scale convention;
- CP holonomy interaction attachment.

The result is bounded, not global. CKM/PMNS charged-current target vertices may
be partially promoted as collider-interface targets where standard
target-current conventions, canonical production basis, and BHSM mixing or
holonomy sources already align. The separate charged boundary response,
neutral kernel, and standalone CP holonomy vertices remain blocked by explicit
missing interaction/basis/scale theorems.

This is not production FeynRules/UFO readiness.
