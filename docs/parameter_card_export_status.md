# BHSM Parameter-Card Export Status v0.2

The parameter-card export records BHSM-derived constants and internal outputs
already present in machine-readable artifacts. These values are not fitted to
empirical data.

Current artifact:

```text
artifacts/BHSM_parameter_card_export_v0_2.json
```

The export is not a production UFO parameter card because the repository does
not yet contain a complete collider-ready 4D Lagrangian, field normalization,
vertex normalization, mass/width convention, or renormalization scheme.
