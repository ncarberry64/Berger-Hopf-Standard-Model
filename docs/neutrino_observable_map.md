# BHSM Neutrino Observable Map

The default observable is a dimensionless effective propagation-mass response.
It is kept distinct from:

- oscillation-sensitive mass differences, whose physical map remains open;
- the electron-neutrino effective-mass comparison, which remains upper-limit
  based by default;
- cosmological mass-sum comparisons, whose map remains open;
- an ordinary static rest-mass interpretation, which is forbidden by the
  controlling ontology.

Electron-neutrino comparisons remain upper-limit comparisons unless a vetted
central mass reference is explicitly supplied.

The ordering policy is `ordering-free`. Dirac/Majorana convention is
`DIRAC_MAJORANA_SECONDARY` unless a later comparison or export theorem requires
it. No ordering or convention is inferred from data.
