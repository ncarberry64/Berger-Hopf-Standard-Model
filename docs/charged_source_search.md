# Charged Source Search

The v1.7 search inventories charged projectors, bridge coefficients, stiffness selectors, eta sources, CKM outputs, CP holonomy, and boundary transport using local machine-readable artifacts.

Artifact-backed values include `Pi_l=1/7`, `Pi_u=2/7`, `Pi_d=4/7`, the charged `beta/kappa` table, and the frozen CKM formulas. Their complete common action normalization is not artifact-backed.

BHSM does not use charged-lepton masses, CKM reference values, PDG values, W calibration, or empirical fitting as theorem inputs.
