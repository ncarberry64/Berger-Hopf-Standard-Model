# BHSM Plain-Language Overview

BHSM means Berger-Hopf Standard Model.

In ordinary language, BHSM treats Standard Model patterns like allowed notes of
an internal geometric instrument. The instrument is a Berger-Hopf internal
geometry. The allowed notes are the particle-sector modes that survive the
repository's rules.

## The Main Idea

The Standard Model has many organized-looking patterns: particle sectors,
generations, mass hierarchies, mixing angles, and a rule that no extra light
particles should appear in the low-energy ledger. BHSM asks whether those
patterns can be reinterpreted as protected modes and overlap patterns of a
fixed internal geometry.

## Particles as Protected Internal Modes

In BHSM language, a particle-sector channel is represented by an internal mode.
The protected kernel contains exactly three sector-labeled physical channels:

- lepton;
- up;
- down.

The release theorem package says this protected kernel is complete according
to the repository ledger.

## No Retuning

The frozen model does not allow changing constants or mode choices after seeing
residuals. The key frozen constants are:

- `a = 1.157054135733433`;
- `S = 0.07957747154594767`.

Changing `a`, `S`, the mode ledger, tolerances, frozen outputs, or the virtual
dressing rule to improve a comparison would break the frozen package.

## Why Mirror and Extra Particles Matter

If a mirror version of a protected state survived, it would behave like an
additional protected particle/mode not present in the frozen ledger. BHSM's
final theorem package classifies mirror candidates as non-protected, lifted,
wrong-orientation, chirality-failing, complement excitations, or forbidden by
the sector ledger. A surviving protected mirror would be a theorem failure.

## What Complete Internal Theorem Package Means

It means the repository ledger marks the internal BHSM theorem chain complete:
operator identification, action uniqueness, projector controls, H_T lower-bound
transfer, index theorem, mirror exclusion, and full H_T theorem are closed in
the release state.

## What It Does Not Mean

It does not mean BHSM has been experimentally confirmed. It does not mean the
scientific community has accepted BHSM. It does not claim QCD confinement, new
particle discovery, or guaranteed correctness.
