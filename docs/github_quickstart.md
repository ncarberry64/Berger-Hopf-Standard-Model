# GitHub Quickstart

## Clone

```powershell
git clone https://github.com/ncarberry64/Berger-Hopf-Standard-Model.git
cd Berger-Hopf-Standard-Model
```

## Python Environment

```powershell
python -m pip install -e .
```

## Run Tests

```powershell
python -m pytest -q
```
