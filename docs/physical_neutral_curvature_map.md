# Physical Neutral Curvature Map

The repository has a dimensionless response and a symbolic map:

```text
R_nu_dimless -> K_neutral,eff=kappa_curv R_nu_dimless.
```

It does not derive `kappa_curv` in `m^-2`. The boundary measure lacks physical
normalization, no transport/support length is available, and the BHSM-specific
profile/embedding and shape-operator values remain open. A dimensionless
response is never treated as physical curvature.
