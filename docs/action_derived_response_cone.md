# Action-Derived Response Cone

The neutral boundary kinetic variation, propagation source, conditional collar
measure, and measurement/interaction support term provide partial action
support for the existing response cone. They do not uniquely derive the
response-magnitude coordinate map, normalization, or complete admissible
boundary data.

Status: `CONDITIONAL_ACTION_DERIVED_RESPONSE_CONE_CANDIDATE`.

An ontology-supported response cone is not the same as a complete-action-derived response cone.

The admissible response cone is currently measurement/interaction-supported and conditionally positive, but its derivation from the complete neutral action remains open.
