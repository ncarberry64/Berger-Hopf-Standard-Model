# Hermitian Charged-Current Action Rule

Status: `CONDITIONAL_HERMITIAN_CHARGED_CURRENT_ACTION_RULE`.

A real or Hermitian action normally pairs an off-diagonal charged-current term with its Hermitian adjoint. The repository has target-level CKM charged-current evidence containing `+ h.c.`.

This is not the same as deriving the BHSM CKM transport space from the normalized action. The charged-current rule, the adjoint-pair selection rule, and the CKM log-transport application are separate gates.

The existence of a Hermitian-conjugate term supports action reality but does not by itself derive CKM transport-space selection.

The Hermitian adjoint-pair channel count is 16, but the CKM exponent remains not derived unless BHSM proves CKM acts on that selected transport space.

No empirical CKM fitting, charged-mass fitting, PDG values, W calibration, neutrino limits, or legacy threshold tables are used as theorem inputs.
