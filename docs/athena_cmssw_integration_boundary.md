# Athena / CMSSW Integration Boundary v0.1.0

Official experiment software integration is out of scope for BHSM v1.0.1.

The repository is not an official ATLAS or CMS software component.

The repository does not claim compatibility with Athena, CMSSW, or experiment
production workflows.

Future integration would require collaboration-specific review, coding
standards, physics validation, generator interfaces, detector simulation
compatibility, and institutional approval.

## Current Status

| Target | Status |
| --- | --- |
| Athena integration | Not ready |
| CMSSW integration | Not ready |
| Official ATLAS software component | No |
| Official CMS software component | No |
| Detector simulation production workflow | Not provided |

| UFO/FeynRules model | Not provided |

## Required Future Work

- Define a complete collider-ready 4D physical Lagrangian.
- Export Feynman rules and a generator model.
- Provide pinned validation targets and comparison plots.
