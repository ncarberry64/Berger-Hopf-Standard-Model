# Notation

- `(k, j)`: Berger-Hopf mode label.
- `q = k - 2j`: Hopf charge.
- `lambda_{k,j}(a)`: Berger scalar spectrum proxy.
- `S = 1/(4 pi)`: universal overlap coefficient.
- `H_T`: topographic stability operator.
- `H_perp`: subspace orthogonal to protected supplied ledger modes.
- `mu_H = 64 pi^5`: dimensionless Hopf-gap target.

Hypercharges use left-handed Weyl notation: `u_c`, `d_c`, and `e_c` denote
charge-conjugate singlet fields.
