# Generation Feature Matrix Construction

For each sector, columns are the three generation labels and rows are the explicit local feature atoms:

- `value_lowest`
- `value_center`
- `value_highest`
- `d_alpha_lowest`
- `d_beta_center`
- `d_gamma_highest`
- `d2_local_center`

Each entry is a Wigner/Hopf jet expression tied to \((k,\ell,n,m)\), not a free fitting symbol.
